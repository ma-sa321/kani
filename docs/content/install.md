## :megaphone: Notice

`kani` runs on [zsh](https://www.zsh.org/), not work on [bash](https://www.gnu.org/software/bash/), now.

## :beer: Homebrew

```sh
brew tap tamadalab/brew
brew install kani
```

## :muscle: Compiling yourself

```sh
go get github.com/tamadalab/kani
```
