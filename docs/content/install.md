## :megaphone: Notice

`kani` runs on [zsh](https://www.zsh.org/), and [bash](https://www.gnu.org/software/bash/) only.
Not work on other shells (`csh`, `ksh`, `fish`, ...).

## :beer: Homebrew (macOS)

```sh
brew tap tamadalab/brew
brew install kani
```

## :muscle: Compiling yourself

```sh
go get github.com/tamadalab/kani
```
