======================================= commit guides ========================================
git status                       shows the status of the current branch of the repository
git branch                       shows the branch list and the current branch
git add <FILE_NAMEs...>          adds specified files to the index (commit targets)
git commit -m "MESSAGE"          commits the changes of the files in the index
git push origin <BRANCH_NAME>    sends the content of the specified branch to the remote repository (origin)
